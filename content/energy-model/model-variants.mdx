As shown in the results below, in order to assess the building performance we have tested the building in six distinct
configurations (variants):

1. **Code-Minimum:** A version which meets but does not exceed the **<VarLink
hrefKey="narrative.energy_code.code_base_url" target="_blank" rel="noopener noreferrer"><Var
k="narrative.energy_code.name" /> / <Var k="narrative.energy_code.zone" /></VarLink>** code minimums for assembly
R-values, window U-values, airtightness and equipment efficiencies. This variant assumes a typical residential
"extract-only" (bath and kitchen fans) ventilation system, and that the building uses gas appliances and equipment for
heating, hot-water and cooking. This version assumes basic electric air-conditioners for cooling and dehumidification. 

2. **Improved Envelope:** This variant improves the window U-values, assembly R-values, and total airtightness of the
building beyond the "code-minimum" levels. This version represents a 'pretty good' envelope. This version retains the basic 
mechanical system and gas heating/hot-water. 

3. **As-Drawn Envelope:** A version which matches the currently **as-drawn** wall, roof, floor, and window performance values. Certain
assumptions have been made for elements which are still being designed (exact glass specification, total thermal-bridge heat loss)
but overall this version captures the as-drawn conditions very closely. This version retains the basic mechanical system and 
gas heating/hot-water. 

4. **Add a Basic ERV:** This variant adds a dedicated fresh-air ventilation system with heat recovery (HRV). This 
system is assumed to be a "basic" HRV with 75% heat-recovery (unit). The dedicated fresh-air system provides higher-quality indoor
air, while minimizing the amount of energy consumption by using passive heat-recovery.

5. **Add Electric Heat Pumps:** A version which upgrades the gas-fueled heating and hot-water generation equipment to more modern
electric [heat-pumps](https://homes.rewiringamerica.org/articles/heating-and-cooling/what-is-a-heat-pump). Heat Pumps are
more efficient than gas and therefore reduces the overall site-energy consumed. This change in equipment does not affect 
the heating or hot-water demanded by the home, but does reduce the energy needed to satisfy that demand. In this variant, all
cooking is still provided by gas appliances. 

6. **Passive House [RECOMMENDED]:** A variant which meets the stringent [PHIUS](https://www.phius.org)
standard's targets for energy consumption, thermal comfort, durability, and health. This variant includes an
upgraded high-performance Energy Recovery Ventilator (ERV) system for fresh-air ventilation, and uses an all-electric HVAC system (heating, cooling, hot-water)
but retains gas for all cooking. This variant assumes that the home achieves an envelope airtightness level in line with the 
Phius standard, and that the thermal-bridging has been almost entirely mitigated. The only change needed to the **as-drawn**
envelope assemblies in order to achieve this level of performance was to increase the sub-slab insulation thickness from 2" to 4.
Note that in this version, winter-humidification is almost eliminated due to the improved moisture-recovery of the high performance
ERV.
