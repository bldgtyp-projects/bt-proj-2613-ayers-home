In addition to the whole-building annual-energy consumption model created using the 
[Passive House Planning Package (PHPP)](https://passivehouse.com/en/home/planning-tools/), the home was 
also assessed using a detailed hourly model for ‘resiliency’ during two test-periods:

1. The home was simulated during a **summer heat-wave event** along with a power-outage which lasts for a full week.
2. The home was simulated during a **winter storm event** along with a power-outage which lasts for a full week.

These ‘stress-test’ simulations evaluate the ability of the building to maintain safe and healthy interior conditions while 
relying only on ‘Passive’ measures alone, as all space conditioning equipment is considered inoperable for these test-periods. 
These test-periods align with the [Phius REVIVE protocol](https://www.phius.org/phius-revive-2024), including all rules 
which define occupancy, weather-conditions, and habitability thresholds. Note in particular that test-period weather 
conditions are established which follow the **Phius REVIVE** requirements.

- [Phius REVIVE 2024 Standard v24.1.1.pdf (7.7 MB)](https://www.phius.org/phius-revive-2024-standard-document)

For all the simulation results shown, the building is configured using the ‘Passive House’ variant attributes for envelope 
elements. See the [Variants section](/energy_model/#model-variants) for more information on the specific building attribute 
values used. For the simulation in this case, the model is split into zones by floor level, and by wing (north vs south) with 
the resilience results being reported for each zone. Note that the simulation includes the 'attic' mechanical spaces, as well as 
the crawlspace in order for their effects on the main spaces to be considered. However these spaces need not achieve the same levels
of thermal resiliency, as they do not have regular occupation.

<div class="btwr-callout btwr-callout--critical">
  <strong>Design implication</strong>
  <span>
    The Summer Heat Index criterion **passes** in the current simulation, however the home does experience several days with interior
    conditions in the 'danger' [>40°C] zone. Passive solar control, improved cross-ventilation, and a battery/generator on-site-powered 
    cooling strategy should all be considered as methods of improving the summer peak-week performance. 
  </span>
</div>
